I describe the vital sections of the python codes below

rxtxpkts.py contains the code for sending and receiving LoRa commands. If a LoRa command is receieved by the LoRa-E5 dev kit in a building, by default, LoRa-E5 dumps the received command to a serial port (in this case, /dev/serial0), the command is then formatted and sent to the Z1 over the /dev/ttyUSB0 port using os.system(f"echo {cmd} > /dev/ttyUSB0")
The command is then sent from the Z1 to a Zn to control an LED

control_led[1|2].py contains the code to turn on or off an LED
